1. 添加新列：ALTER TABLE courses ADD COLUMN time_start INTEGER...
2. 迁移数据：UPDATE courses SET time_start = time...
3. 删除旧列：ALTER TABLE courses DROP COLUMN time...

我现在重新建一个数据库，请修改.sql文件，可直接一键生成，并且无上述问题

AI：
修改supabase_schema.sql，在文件开头添加：
DROP TABLE IF EXISTS course_modifications CASCADE;
DROP TABLE IF EXISTS courses CASCADE;
DROP TABLE IF EXISTS todos CASCADE;
DROP TABLE IF EXISTS semester_settings CASCADE;
DROP TABLE IF EXISTS time_slots CASCADE;

这样可以先删除旧表，再创建新表，避免字段冲突。
